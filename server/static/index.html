How our system works


Photo detection

Our system has object detection when picture is uploaded for verification. It detects if there is a person on the picture or something else.

It can also detect age of the person based on picture and compare it to the age provided on the passport.

Error level analysis

Error Level Analysis is the analysis of compression artifacts in digital data with lossy compression. When used, lossy compression is normally applied uniformly to a set of data, such as an image, resulting in a uniform level of compression artifacts.

Our system can detect this kind of changes.

MRZ checking

Most travel passports worldwide have a special machine-readable zone (MRZ), which is usually at the bottom of the identity page at the beginning of a passport.Machine-readable passports enable faster processing of arriving passengers by immigration officials, and greater accuracy than manually read passports, as well as faster data entry and data matching against immigration databases and watchlists.

Our system can check if mrz code and checks correctness of it.

Blur picture detection

If the provided passport picture is blured, it's impossible to read data from it.

System can detect if it's blured or not. If it is, there is no need to continue checking other data, and verification stops.

EXIF checking

Exchangeable image file format(EXIF) is a standard that specifies the formats for images, sound, and ancillary tags used by digital cameras (including smartphones), scanners and other systems handling image and sound files recorded by digital cameras.

If there was made some modification to the original picture(photoshoped,etc.), our system can detect it and stop verification

Grayscale detection

In photography and computing, a grayscale or greyscale digital image is an image in which the value of each pixel is a single sample, that is, it carries only intensity information.

Unlike other softwares, if provided passport picture is grayscale, our system is capable of handling this situation and extracting data from it.
